I asked Gemini "does el capitan use linux"

Yes, El Capitan absolutely uses Linux as its operating system.1


More specifically, it uses TOSS (Tri-Lab Operating System Software), which is a custom Red Hat Enterprise Linux (RHEL) distribution developed by Lawrence Livermore National Laboratory (LLNL) and the other National Nuclear Security Administration (NNSA) labs (Los Alamos and Sandia).2


The choice of Red Hat Enterprise Linux as the base for TOSS provides a robust, stable, and widely supported foundation, while TOSS itself allows for specialized customizations and optimizations necessary for managing an exascale supercomputer and its unique hardware (like the AMD MI300A APUs and HPE Slingshot interconnect). This standardization also helps streamline administration and provides a consistent user experience across different systems within the NNSA complex.
